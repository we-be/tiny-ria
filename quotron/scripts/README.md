# Scripts Directory

This directory contains utility scripts for setting up and configuring Quotron services.

## Available Scripts

### API Setup

- `api-setup/alpha_vantage_setup.sh`: Script for setting up Alpha Vantage API keys and testing connectivity

## Usage

Most functionality previously contained in bash scripts has been migrated to the consolidated Go CLI. Please use the main `quotron` command instead.

For more information on using the CLI, run:
```bash
./quotron help
```
